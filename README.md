# 🛍️ Mall Customer Segmentation Analysis

A Machine Learning project that analyzes mall customer data and groups customers into different segments using clustering techniques. This helps businesses understand customer behavior and improve marketing strategies.

---

## 🚀 Features

- 📊 Customer segmentation using ML algorithms  
- 🧠 Uses clustering techniques (K-Means / t-SNE)  
- 📈 Data visualization for better insights  
- 🌐 Simple web interface for interaction  
- ⚡ Real-time analysis results  

---

## 🛠️ Tech Stack

- **Frontend:** HTML  
- **Backend:** Python (Flask)  
- **Machine Learning:** Scikit-learn, Pandas, NumPy  
- **Visualization:** Matplotlib, Seaborn  

---

```
mall-customer/
```

## ⚙️ Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/Durgamvani-184/mall-customer.git
cd mall-customer
```

### 2. Install Dependencies
```bash
pip install pandas numpy scikit-learn matplotlib seaborn flask
```

### 3. Run the Application
```bash
python app.py
```

### 4. Open in Browser
```
http://127.0.0.1:5000/
```
